Course Dalvin

Android Developer

Android Developer adalah orang yang bertanggung jawab untuk mengembangkan dan membangun perangkat lunak berbasis mobile menggunakan sistem operasi Android. Jadi Di kelas ini kamu belajar tentang apa serba serbi Android
IOS Developer

IOS Developer adalah salah satu pekerjaan terkait dengan pengembangan aplikasi mobile (seluler) dengan menggunakan sistem operasi iOS Apple. Jadi tugas yang dimiliki adalah membuat aplikasi yang dapat dijalankan pada sistem operasi tersebut.

Back-End Developer

Back-End Developer adalah seseorang yang bertanggung jawab untuk mengelola server website (server-side). Artinya, seorang backend developer perlu memastikan website bekerja dengan baik dalam kaitan pertukaran data dari browser ke server. Selain itu, backend developer juga menangani database dan membuat alur website yang terstruktur.
Front-End Developer

FrontEnd adalah bagian dari pengembangan web yang mengkodekan dan membuat FrontEnd depan situs web, yang merupakan fitur yang dapat dilihat dan diakses secara langsung oleh pengguna akhir atau klien.

Machine Learning

Machine Learning Developer adalah pakar dalam menggunakan data untuk model pelatihan. Model-model tersebut kemudian digunakan untuk mengotomatisasi proses seperti klasifikasi gambar, pengenalan suara, dan perkiraan pasar. Sering kali ada penggabungan dengan peran data scientist atau artificial intelligence (AI) engineer.
Design Class

Desain Class adalah membuat object dengan bantuan Software seperti Adobe Ilustrator, Photoshop dan Corel Draw
Office Application

Office Application adalah Class belajar tentang Microsoft Office seperto Word, Excel dan Power Point

Daftar Biaya Belajar
Pilih paket langganan sesuai budget anda dan sebagai investasi belajar yang sesuai dengan kebutuhan Anda.
Paket Langganan Belajar
  • Akses penuh ke seluruh kelas di Dalvin.
  • Bimbingan dan bedah kode (code review) dari para expert developer terhadap tugas submission Anda.
  • Sertifikat kelulusan jika Anda berhasil menyelesaikan kelas berikut submission proyek akhir dengan baik.
  • Konsultasi sepuasnya di forum diskusi.
  • Bayar putus. Tak ada kewajiban untuk melanjutkan langganan. Tak ada tagihan rahasia.

Tips Pencegahan

Menggunakan Masker

Gunakanlah Masker Standart Kesehatan untuk mencegah virus masuk ke pernapasan.

Mencuci Tangan

Rutin mencuci tangan dengan air dan sabun atau hand sanitizer yang mengandung alkohol minimal 60%.

Menjaga Jarak

Terapkan physical distancing, yaitu menjaga jarak minimal 1 meter dari orang lain, dan jangan dulu ke luar rumah kecuali ada keperluan mendesak.

Tetap Di Rumah

Jika tidak ada keperluan mendesak harap tetap dirumah saja.
